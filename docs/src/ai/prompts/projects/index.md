# AI 项目开发提示词专栏

本专栏收录适用于实际 AI 项目开发的精选提示词模板，涵盖前端、后端、移动端等多个场景。

## 专栏目的

- **沉淀经验**：将项目中反复验证的提示词结构化保存
- **快速复用**：通过模板复制快速生成高质量代码
- **团队共享**：统一团队的提示词规范和最佳实践

## 内容分类

| 分类 | 说明 |
|------|------|
| 前端开发 | React、Vue、组件、样式、状态管理等 |
| 后端开发 | API、数据库、认证、微服务等 |
| 移动端 | UniApp、Flutter、响应式等 |
| 工具类 | 脚本、CI/CD、部署、监控等 |
| 文档生成 | README、API 文档、注释等 |

## 使用方式

### 1. 直接使用

复制文档中的提示词模板，根据实际需求修改变量后使用。

### 2. 自定义模板

```bash
# 保存为自定义模板
claude-code prompt create --name my-template
```

### 3. 迭代优化

根据项目实际情况调整提示词结构，不断优化生成效果。
